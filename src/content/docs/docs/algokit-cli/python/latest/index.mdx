## Installation

<Tabs>
  <TabItem label='macOS / Linux'>
    ```bash curl -fsSL https://cli.algokit.io/install.sh | bash ```
  </TabItem>
  <TabItem label='Windows'>```powershell irm https://cli.algokit.io/install.ps1 | iex ```</TabItem>
</Tabs>

> The install script handles installing `uv` and Python automatically. Legacy installation methods via `brew`, `winget`, and `snap` are also available — see the [official installation guide](https://github.com/algorandfoundation/algokit-cli#install).

## Quick Start

```bash
# Verify installation
algokit --version

# Create a new project from a template
algokit init

# Start a local Algorand network
algokit localnet start

# Explore the network with lora
algokit explore
```
