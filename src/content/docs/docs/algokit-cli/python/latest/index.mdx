## Installation

<Tabs>
  <TabItem label='macOS'>```bash brew install algorandfoundation/tap/algokit ```</TabItem>
  <TabItem label='Windows'>```bash winget install AlgoKit ```</TabItem>
  <TabItem label='pipx (any OS)'>```bash pipx install algokit ```</TabItem>
</Tabs>

## Quick Start

```bash
# Verify installation
algokit --version

# Create a new project from a template
algokit init

# Start a local Algorand network
algokit localnet start

# Explore the network with lora
algokit explore
```

## Features

  <Card title='Init' icon='rocket'>
    Quickly scaffold new Algorand projects from official templates or community templates.

  <Card title='LocalNet' icon='laptop'>

  <Card title='Generate' icon='document'>
