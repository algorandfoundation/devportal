# AlgoKit CLI

AlgoKit CLI is a command-line tool for Algorand development that simplifies common tasks.

## Installation

```bash
pipx install algokit
```

## Quick Start

```bash
# Initialize a new project
algokit init

# Start LocalNet
algokit localnet start

# Compile smart contracts
algokit compile
```

## Features

- Project scaffolding with templates
- LocalNet management
- Smart contract compilation
- Code generation
