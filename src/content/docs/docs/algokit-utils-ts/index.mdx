# AlgoKit Utils TypeScript

A set of TypeScript utilities that make it easier to build solutions on Algorand.

## Installation

```bash
npm install @algorandfoundation/algokit-utils
```

## Quick Start

```typescript
import { AlgorandClient } from '@algorandfoundation/algokit-utils'

const algorand = AlgorandClient.defaultLocalNet()
```

See the [Quick Start guide](/docs/algokit-utils-ts/quick-start/) to get started.
