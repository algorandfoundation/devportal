      </td>
      <td>
        <p>
          This example demonstrates how to perform an atomic swap of ALGO for ASA between two
          parties. In an atomic swap:
        </p>
        <ul>
          <li>Party A sends ASA to Party B</li>
          <li>Party B sends ALGO to Party A</li>
          <li>Each party signs ONLY their own transaction</li>
          <li>Signatures are combined and submitted together</li>
          <li>Both transfers succeed or both fail (atomicity)</li>
        </ul>
        <p>
          Key difference from regular atomic groups: different parties sign different transactions.
        </p>
        <p>Prerequisites:</p>
