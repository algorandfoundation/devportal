LORA stands for live on-chain resource analyzer, and is a great tool for interacting with the blockchain without having to build a custom front end yourself. It can be run locally with AlgoKit or you can use the hosted version at [https://lora.algokit.io](https://lora.algokit.io). 

It provides a basic GUI allowing you to take actions on localnet, testnet, or mainet.
