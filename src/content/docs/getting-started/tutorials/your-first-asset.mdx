**Hello, Developer! Welcome to your first asset on Algorand!**

This guide will walk you through two ways of creating your first **Algorand Standard Asset (ASA)** on LocalNet.

By the end of this guide, you will:

- Configure an ASA using AlgoKit
- View your asset on chain using LORA

If you want to know more about what ASAs are check out the documentation here:

<LinkCard title='Opt-in' href='/concepts/assets/overview' description='What are ASAs?' />

Let's begin!

#### Creating an asset with LORA

<Steps>
    1. Follow the steps in [Getting ready to build](/getting-started/tutorials/get-ready-to-build) to start LocalNet
    
    2. Follow the steps in [Using LORA](/getting-started/tutorials/using-lora) to connect a test wallet funded with Algo.

    3. Navigate to Txn Wizard.

    4. Click `Add Transaction`.

        <Figure src={LORAaddTransaction} alt='Add Transaction' />
 

    5. Choose `Asset create` from the dropdown and fill out all your desired fields.
        
        For this example we'll use the following basic config:
        
        <Figure src={LORAassetConfig} alt='Asset Configuration' />

        *Note that the creator field will be auto-populated for you with the wallet address you've connected to the site.*
        
        For more information on the optional fields like Manager, Reserve, Freeze, and Clawback check out the [ASA documentation](/concepts/assets/overview). 

    6. Click `Add` and you will then see the Asset Config transaction added to the Transaction Group. Click `Send` to submit the transaction. 

        <Figure src={LORAsubmitCreateAssetTransaction} alt='Submit Transaction' />
    
    7. You've now successfully created an asset using LORA! Now we encourage you to go back and learn how to create an asset in [TypeScript](/getting-started/tutorials/your-first-asset#choose-lora-or-a-programming-language).

<TabItem label="TypeScript" icon="seti:typescript">

<Steps>

    1. Follow the steps in [Getting ready to build](/getting-started/tutorials/get-ready-to-build) to initialize a new AlgoKit project and start LocalNet

    2. Create a new file where we will write the script that will create your new asset. You can create it where ever you'd like, but in this example we've created it in the root directory and called it `first-asset.ts`

        <Figure src={TScreateAssetFile} alt='File Structure' />

    3. Write the script that will create the asset. 

        Each of the following sub-sections will add code to the file you just created:

        a. Initialize the client and create the test account that will create the asset

        ```ts showLineNumbers=false frame=none
        import { AlgorandClient, algo } from "@algorandfoundation/algokit-utils";

        async function main(): Promise<void> {
        console.log("🚀 Creating your first asset...");

        const algorand = AlgorandClient.defaultLocalNet(); //create client to interact with your LocalNet

        const alice = await algorand.account.fromEnvironment("ALICE", algo(100)); //ALICE is a standard default test account

        console.log(`Alice: ${alice.addr}`);
        ```

        b. Pass in the parameters of the [assetCreate()](/algokit/utils/typescript/asset/#creation) method

        ```ts showLineNumbers=false frame=none
        const result = await algorand.send.assetCreate({
            //Mandatory fields
            sender: alice.addr,
            decimals: 6, //Number decimal places a whole unit can be broken up into
            total: 1_000_000_000n, //Number whole units + decimal places, ie. 1,000
            defaultFrozen: false,

            //Optional asset config fields
            assetName: 'My First ASA',
            unitName: 'MFA',
            manager: alice.addr,
            reserve: alice.addr,
            freeze: alice.addr,
            clawback: alice.addr,
            url: 'https://algorand.co',
        });

        //Capture the unique asset ID of the asset after it's created
        const assetId = result.assetId; // This is automatically assigned to each ASA as it's created

        console.log(`Asset created with ID: ${assetId}`);
        }
        ```

        c. Call the main function
        ```ts showLineNumbers=false frame=none
        main().catch(console.error); //Calls the function when you run the script
        ```

        d. The final script should look ike this:
            ```ts showLineNumbers=false frame=none
            import { AlgorandClient, algo } from "@algorandfoundation/algokit-utils";

            async function main(): Promise<void> {
            console.log("🚀 Creating your first asset...");

            const algorand = AlgorandClient.defaultLocalNet();

            const alice = await algorand.account.fromEnvironment("ALICE", algo(100));

            console.log(`Alice: ${alice.addr}`);

            const result = await algorand.send.assetCreate({
                //Mandatory fields
                sender: alice.addr,
                total: 1_000_000_000n, 
                decimals: 6, 
                defaultFrozen: false,
            
                //Optional asset config fields
                assetName: 'My First ASA',
                unitName: 'MFA',
                manager: alice.addr,
                reserve: alice.addr,
                freeze: alice.addr,
                clawback: alice.addr,
                url: 'https://algorand.co',
            });
            
            //Capture the unique asset ID of the asset after it's created
            const assetId = result.assetId;
            
            console.log(`Asset created with ID: ${assetId}`);
            }

            main().catch(console.error);
            ```
    
    4. Now run your script and take a look at the logs. In your terminal navigate to your project directory and run:

        ```shell showLineNumbers=false frame=none
        npx tsx first-asset.ts
        ```

    5. Your console log should look something like this:
        <Figure src={TScreateAssetLog} alt='Create Asset Logs' />

    6. You've successfully created a new asset programmatically! To view it on the blockchain you can use [LORA](/algokit/lora/overview). In your terminal type the following command to open it in the browser:

        ```shell showLineNumbers=false frame=none
        algokit explore
        ```
</Steps>

#### View Your Asset in LORA

You can now view your asset onchain using LORA. 

<Steps>
    1. If you haven't already, open Lora and connect your wallet

        <Figure src={LORAConnectWallet} alt='Lora home screen' />

    2.  Click on **Accounts** and select Alice

        <Figure src={LORAConnectAlice} alt='Lora connect wallet modal' />

    3.  With Alice connected you cna now click `view` to see the account's information

        <Figure src={LORAassetCreatViewAlice} alt='View Alice' />

    4. YOu will see a history of all the transactions from this account, and if you scroll to the **Assets** section, you should see your newly created asset listed.

        <Figure src={LORAassetCreatorBalance} alt="Creator's Balance"/>

    4.  Click the Asset ID to view the asset. You will see the configuration details you created that asset with, as well as a history of all the transactions for that asset. 

        <Figure src={LORAassetCreateAssetinfo} alt='Asset details' />
</Steps>
